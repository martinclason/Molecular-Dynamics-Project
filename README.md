# How to run the software:

## On your own machine:
Install ASE:

$ pip install ase

Install ASAP:

$ pip install asap3

Run main.py:

$ python3 main.py


## On LiU Linux lab computer:

Install ASE and ASAP Python modules:

$ source/TFYA74/software/bin/init.sh

Run main.py:

$ python3 main.py

Run ale:
$ ale

Without asap and special config
```
$ ale --no-asap my_config.yaml
```
